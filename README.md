## 🚀 Cara Menjalankan

1. Clone repository:
git clone https://github.com/sefianasya/biodata-sefia.git cd biodata-project

2. Install dependencies:
npm install

3. Jalankan server:
npm run serve

4. Buka di browser:
http://localhost:8080/biodata
